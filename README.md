# skycast - 天气仪表盘

skycast 是一个基于和风天气 API 开发的现代化天气仪表盘，提供实时天气信息、预报和各种生活指数，帮助用户更好地了解天气状况和规划活动。

## 项目特点

- 🌤️ **实时天气信息**：显示当前温度、天气状况、体感温度等核心数据
- 📅 **5天天气预报**：提供未来5天的天气趋势预测
- ⏰ **24小时逐小时预报**：详细的每小时天气变化
- 📊 **生活指数**：包括运动、洗车、穿衣、舒适度、紫外线、感冒等多种指数
- 📈 **温度趋势图**：直观展示温度变化趋势
- ⚠️ **天气预警信息**：及时获取官方发布的天气预警
- 📱 **响应式设计**：适配不同屏幕尺寸的设备
- 🎨 **精美的UI设计**：现代化的界面和流畅的动画效果

## 技术栈

- **前端框架**：原生 JavaScript (ES6+)
- **样式框架**：CSS3
- **图表库**：Chart.js
- **API**：和风天气 API
- **工具库**：Font Awesome, QWeather Icons

## 安装和使用

### 本地运行

1. **克隆项目**
   ```bash
   git clone https://github.com/hotier/weatherwave.git
   cd weatherwave
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **启动本地服务器**
   ```bash
   python -m http.server 8000
   ```

4. **访问应用**
   打开浏览器访问 `http://localhost:8000`

### 配置

要使用和风天气 API，您需要在 `script.js` 文件中配置您的 API 密钥：

```javascript
// 和风天气API配置
const API_CONFIG = {
    host: 'api.qweather.com', // 和风天气官方API主机
    key: '您的API密钥'
};
```

您可以在 [和风天气开发者平台](https://dev.qweather.com/) 注册并获取免费的API密钥。

## 功能介绍

### 当前天气
- 实时温度、天气状况、体感温度
- 湿度、风速、风向、气压等详细数据
- 日出日落时间
- 云量、能见度、降水量等信息

### 天气预报
- 5天天气预报，包含最高/最低温度
- 天气状况和降水概率
- 24小时逐小时预报，展示每小时的天气变化

### 生活指数
- 运动指数：提供运动建议和注意事项
- 洗车指数：根据天气状况推荐洗车时间
- 穿衣指数：提供穿衣建议
- 舒适度指数：评估天气舒适度
- 紫外线指数：提示紫外线强度和防护建议
- 感冒指数：提供感冒预防建议

### 其他功能
- 温度趋势图：可视化展示温度变化
- 天气预警：显示官方发布的天气预警信息
- 分享功能：生成天气分享卡片
